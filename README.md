# Travel Buddy 🗺

Flutter Android app to **Discover · Save · Share** travel destinations, with gamification (XP, levels, achievements, streaks) to make exploration a game.

Built for the *Mobile Application Development* course — final submission (P4, June 2026).

## Build from source

### Prerequisites
- Flutter SDK ≥ 3.29 ([install guide](https://docs.flutter.dev/get-started/install))
- Android SDK (API 26+) with `ANDROID_HOME` set
- A Google Maps API key (free tier, [console.cloud.google.com](https://console.cloud.google.com) → "Maps SDK for Android")

### Steps
```bash
# 1. Clone
git clone https://github.com/khaledakidi/travel-buddy.git
cd travel-buddy

# 2. Add your Google Maps API key
#    Edit android/app/src/main/AndroidManifest.xml and replace:
#      android:value="YOUR_GOOGLE_MAPS_API_KEY"
#    with your real key.

# 3. Install dependencies
flutter pub get

# 4. Run on a connected Android device or emulator
flutter run
```

> Note: the repo ships with `lib/`, `pubspec.yaml`, and the manifest. The Android Gradle scaffolding is generated by `flutter create .` (CI does this automatically; locally Flutter does it on first `flutter run`).

## Features (vs. proposal)

### Main features ✅
- Google Maps with current location + colored category pins
- Category filters: All / Food / Drinks / Attractions / Activities / Nightlife / Shopping / Nature
- Search bar (filters by name + address as you type)
- Three view modes: Map / Grid / List
- Save favorites with heart button (SQLite via `sqflite`)
- Place detail bottom sheet (rating, hours, price, Mark Visited, Share, Directions)
- Student discount badges with discount text
- Travel tips with tag filters (Transport / Savings / Food / Activities / etc.)

### Gamification ✅
- **XP:** +5 per save, +5 per share, +10 per visit, +2/day streak bonus
- **Levels:** L1 (0 XP) → L2 (50) → L3 (150) → L4 (300) → L5 (500)
- **6 achievement badges:** First Visit · 5 Saved · Foodie · Culture Vulture · Night Owl · Budget Pro
- **Streak tracking** with consecutive-day bonus XP
- **Profile stats:** Saved · Visited · Shared (with streak chip 🔥 on avatar)
- **Visited checklist:** green check on saved cards

### Settings ✅
- Student Mode toggle (persists via `SharedPreferences`)
- Notifications toggle
- Language, appearance preferences

## Project structure

```
lib/
├── main.dart                    ← App entry + theme
├── app_state.dart               ← Provider state (filters, saves, badges)
├── data/
│   ├── db/app_database.dart     ← SQLite CRUD + XP/streak logic
│   └── model/place.dart         ← Place model + 16 demo Istanbul places
├── screens/
│   ├── splash_screen.dart       ← 2s splash with logo
│   ├── main_screen.dart         ← 4-tab bottom nav
│   ├── explore_screen.dart      ← Map + search + filters + 3 views
│   ├── saved_screen.dart        ← Favorites + dismiss-to-delete
│   ├── tips_screen.dart         ← Travel tips with tag filters
│   └── profile_screen.dart      ← XP bar, badges, settings
└── widgets/
    ├── place_card.dart          ← Grid + list card variants
    └── place_detail_sheet.dart  ← Modal bottom sheet
```

## Tech stack

| Layer | Package | Why |
|---|---|---|
| Framework | Flutter + Dart | Cross-platform UI |
| Maps | `google_maps_flutter` | Interactive map + markers |
| Location | `geolocator` + `permission_handler` | GPS + permission flow |
| Storage | `sqflite` | Saves, visits, XP, streak, share count |
| State | `provider` | Filter / save / achievement state |
| Sharing | `share_plus` | Android share sheet |
| Routing | `url_launcher` | Open Google Maps for directions |
| Prefs | `shared_preferences` | Persisted settings |
